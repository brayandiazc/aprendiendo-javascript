# 08 · DOM y eventos

Cómo JavaScript da vida a las páginas web: acceder y modificar el DOM, reaccionar a eventos,
manipular atributos y guardar datos en el navegador.

> Los ejemplos de este módulo se ejecutan en el **navegador** (no en Node). Abre la consola
> de desarrollador con `F12`. Hay ejemplos HTML listos para abrir en [`ejemplos/`](./ejemplos).
