# Ejemplos del módulo DOM

Ejemplos **autocontenidos** para practicar el DOM. Cada archivo `.html` incluye su propio
HTML, CSS y JavaScript: solo ábrelo en el navegador (doble clic o "Abrir con…") y juega con
la consola de desarrollador (`F12`).

| Ejemplo                                              | Demuestra                                            |
| ---------------------------------------------------- | ---------------------------------------------------- |
| [01-seleccion-y-eventos.html](./01-seleccion-y-eventos.html) | Seleccionar elementos y responder a eventos `click`. |
| [02-elementos-dinamicos.html](./02-elementos-dinamicos.html) | Crear y eliminar elementos del DOM en tiempo real.   |
| [03-localstorage.html](./03-localstorage.html)       | Guardar y recuperar datos con `localStorage`.        |
