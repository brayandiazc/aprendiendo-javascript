# Proyectos

Proyectos prácticos que integran lo aprendido en los módulos. Cada proyecto avanza por
etapas para que veas cómo el mismo problema se resuelve aplicando conceptos cada vez más
avanzados.

## Conversor de monedas

Una pequeña aplicación que convierte montos entre monedas, construida paso a paso:

1. [Variables](./conversor-monedas/01-conversor-monedas-variables.js) — tasas y cálculo básico.
2. [Inputs](./conversor-monedas/02-conversor-monedas-inputs.js) — entrada del usuario.
3. [Condicionales](./conversor-monedas/03-conversor-monedas-condicionales.js) — validación de datos.
4. [Ciclos](./conversor-monedas/04-conversor-monedas-ciclos.js) — múltiples conversiones.
5. [Funciones](./conversor-monedas/05-conversor-monedas-funciones.js) — refactor con funciones.

Descripción completa: [conversor-monedas.md](./conversor-monedas.md).

## Buscador de usuarios

Proyecto **integrador**: descarga usuarios de una API pública y los filtra por nombre en
tiempo real. Combina DOM, `fetch`, asincronía y manejo de errores.

- App: [buscador-usuarios/index.html](./buscador-usuarios/index.html) (ábrelo en el navegador).
- Lógica comentada: [buscador-usuarios/app.js](./buscador-usuarios/app.js).
- Descripción completa: [buscador-usuarios.md](./buscador-usuarios.md).
