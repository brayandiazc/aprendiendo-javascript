# Convenciones

Esta carpeta documenta **cómo trabajamos** en "Aprendiendo JavaScript": las reglas
transversales para escribir, nombrar y verificar el contenido del curso.

## Convenciones incluidas

| Convención                                             | Tema                                            |
| ------------------------------------------------------ | ----------------------------------------------- |
| [estilo-de-contenido.md](estilo-de-contenido.md)       | Estructura de una lección, tono y formato       |
| [nomenclatura.md](nomenclatura.md)                     | Nombres de módulos, lecciones y archivos        |
| [verificacion-de-ejemplos.md](verificacion-de-ejemplos.md) | Cómo se ejecutan y verifican los ejemplos    |
| [quality-tooling.md](quality-tooling.md)               | Markdownlint, formato y revisión de enlaces     |
| [ai-agents.md](ai-agents.md)                           | Trabajo con agentes de IA                       |

## Agregar una convención

Copia [`_template.md`](_template.md), renómbralo en `kebab-case` y documenta el nuevo
tema. Añádelo a la tabla de arriba.
