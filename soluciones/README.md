# Soluciones a los ejercicios

Esta carpeta contiene las soluciones de los ejercicios **como archivos ejecutables**, para
que puedas correrlas y experimentar con ellas. Cada solución ya aparece explicada dentro de
la lección de ejercicios de su módulo; aquí las tienes listas para ejecutar.

> Intenta resolver los ejercicios por tu cuenta antes de mirar la solución. Aprender es
> equivocarse y volver a intentarlo. 🙂

## Cómo ejecutarlas

Necesitas [Node.js](https://nodejs.org) (versión 18 o superior para el módulo de fetch):

```bash
node soluciones/07-poo.js
node soluciones/09-asincronia.js
node soluciones/10-fetch-y-apis.js          # requiere internet
node soluciones/11-manejo-de-errores.js
node soluciones/12-modulos-es/main.mjs
node soluciones/13-strings-numeros-fechas.js
```
